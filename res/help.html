Help

  • Type the Danbooru/Yande.re tags you want to search into the search box. Make sure to use one tag per line.
  • Danbooru's limit of 2 tags does not apply - you may use as many tags as you want. You may also exclude as many tags as you want (using "-whatever", for example). However, complex queries like "or" statements only have limited support. Meta tags, with the exception of "rating:_", also have limited support.
  • There is an autocomplete, so if you partially type a tag, it will give you suggestions. Some common tags are cached for instant suggestions, and more obscure ones will appear afterwards. Click the "Clear Cache" button to clear the cache, and it will not be regenerated until you visit this site again. Even if you do not clear the cache, it refreshes by itself every 7 days.
  • Once the images are loaded, tap/click on any image to view it in full resolution.
  • If you experience a bug (especially with autocomplete/searching), try clicking the "Clear Cache" button and it might fix it. If not, please email me at estilpossiblededeveniruneetoile (at) yahoo.com
