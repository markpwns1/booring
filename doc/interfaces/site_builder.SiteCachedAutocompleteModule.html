Interface SiteCachedAutocompleteModule

Represents an autocomplete system that queries a URL once to obtain a local cache, and uses that cache for all autocomplete operations. Note that this ONLY works for caches of the format: 

3`firsttag`alias` 0`secondtag` 4`thirdtag`

Used in Konachan for example.

interface SiteCachedAutocompleteModule {
    delay?: number;
    minLength?: number;
    summaryUrl: string;
    typeToEnum: {
        [key: string | number]: TagType;
    };
    version: string;
}
